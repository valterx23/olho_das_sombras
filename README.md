# 👁️ Olho das Sombras 👁️
> Framework educacional em Python para apoio a estudos de Pentest no Kali Linux.

---

## 📌 Sobre o Projeto

O **Olho das Sombras** é um projeto educacional desenvolvido com o objetivo de apoiar o aprendizado em **Segurança da Informação e Pentest**, utilizando Python para automatizar e centralizar chamadas a ferramentas amplamente conhecidas no meio de segurança.

O foco é o uso **exclusivo em ambientes controlados**, como:
- Máquinas virtuais
- Laboratórios locais
- Ambientes onde há autorização formal para testes

---

## ⚠️ Aviso Legal

Este projeto tem **finalidade exclusivamente educacional**.

- Utilize **apenas em ambientes próprios ou autorizados**.
- O repositório **não tem a intenção de facilitar atividades ilegais**.
- O autor **não se responsabiliza por qualquer uso indevido deste software**.

O conhecimento em segurança deve ser usado para **proteger sistemas**, não para prejudicá-los.
